# Queue Ticket Generator

A simple application to generate and manage queue tickets for service queues.

## Features

- Generate sequential queue tickets
- Display current ticket number
- Reset queue counter
- Simple web interface

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Alac1996/queue-ticket-generator.git
cd queue-ticket-generator
```

2. Install dependencies:
```bash
npm install
```

3. Start the application:
```bash
npm start
```

## Usage

Open your browser and navigate to `http://localhost:3000` to access the queue ticket generator.
